# Quiz Generator
#### Video Demo:  https://youtu.be/t4vbNwse-zM
#### Description:
This project asks the user for an input of topic for the quiz.Topic shloud be from selected topics (case insensitively)
Then,It asks user for the number of questions.
Then,It generates the quiz on the given topic with specified number of questions.

It uses wikipedia module to fetch the content of the topic and then uses nltk mmodule to generate sentences.
It uses random module to randomly  create options and blanks.
